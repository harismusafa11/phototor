# Phototor Studio — Free Online Photo Editor & PSD Web Tool

Phototor Studio is a powerful, free browser-based professional photo editor. Edit multi-layer designs, PSD files online, crop with Slice Tool & Grid Cutter, auto-remove backgrounds with AI, apply layer styles, and perform photo retouching directly in your browser without any installation.

## 🚀 Key Features

- **Multi-Layer Canvas Workspace**: Manage unlimited layers, blending modes, opacity, and layer groups.
- **PSD Support & Multi-Format Export**: Export to PSD, PNG Transparent, JPEG, and WEBP formats.
- **Slice Tool & Grid Cutter Pro**: Slice layouts into precision grids for web asset exports in ZIP format.
- **AI Background Remover & 4K Upscaler**: Auto background removal and super-resolution upscaling.
- **Layer Styles (FX)**: Drop Shadow, Inner Shadow, Stroke, Outer Glow, Inner Glow, Bevel & Emboss, and Color Overlay.
- **Professional Color Adjustments**: Curves, Levels, Hue/Saturation, Color Balance, Channel Mixer, Photo Filter, and Real-time Histogram.

## 🛠️ Run Locally

**Prerequisites:** Node.js (v18+)

1. Clone or download the repository:
   ```bash
   git clone https://github.com/phototor/phototor-studio.git
   cd phototor-studio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
